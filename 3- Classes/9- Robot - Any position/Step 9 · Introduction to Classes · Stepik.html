Robot: Any position

A robot stands on the playfield. The position of a robot on the field is described by two coordinates: X and Y. The X-axis goes from left to right, the Y-axis -- from bottom to the top. (Remember how did you draw the function graph in school?)

Initially, the robot is at some position in the field. Also, we know where the robot looks: "UP", "DOWN", "RIGHT" or "LEFT". Your task -- move the robot to a given point of the playfield.

A robot is described by the class Robot. You can use the following functions (the implementation is not known to you):

    def get_direction(self) -> str:
        # current direction of a look

    def get_x(self) -> int:
        # current X coordinate

    def get_y(self) -> int:
        # current Y coordinate

    def turn_left(self) -> None:
        # turn 90 degrees counterclockwise

    def turn_right(self) -> None:
        # turn 90 degress clockwise

    def step_forward(self) -> None:
        # one step towards the direction of a look
        # for one of its step robot changes one of its coordinates by one

A robot can look four ways: "UP", "DOWN", "RIGHT", "LEFT".

How it looks like:

Task: create a function move_robot(r: Robot, to_x: int, to_y: int) -> None that given a robot and specific positions to_x and to_y on a playfield, moves it to this position.

Example of using:

r = Robot(10, 2, "RIGHT")
move_robot(r, 2, -1)
print(r.get_x())  # should print 2
print(r.get_y())  # should print -1
