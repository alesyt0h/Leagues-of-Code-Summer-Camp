2D Points (Part II)

Consider a 2D plane. We'll add and remove points to/from this plane.

You need to implement 3 functions and a way to store data.

• Function add_point(x: int, y: int) -> None to add a new point with coordinates (x, y) to the plane. It is guaranteed that there is no point with such coordinates at the time of the function call.

• Function del_point(x: int, y: int) -> None to remove a point with coordinates (x, y) from the plane. It is guaranteed that there is a point with such coordinates at the time of the function call.

• Function closest() -> int to find the squared Euclidean distance between the two closest points on the plane. It is guaranteed that there are at least two points at the time of the function call. The squared Euclidean distance between two points (x1,y1)(x_1, y_1)(x1​,y1​) and (x2,y2)(x_2, y_2)(x2​,y2​) equals to (x1−x2)2+(y1−y2)2(x_1 - x_2)^2 + (y_1 - y_2)^2(x1​−x2​)2+(y1​−y2​)2.

• Function area() -> int to find the maximum area of a rectangle with sides parallel to the coordinate axes and whose vertices are from the current points. If no such rectangle exists, then the function should return zero.

 

Example of usage:

add_point(0, 0)
add_point(2, 0)
add_point(0, 4)
print(closest()) # should print 4
print(area()) # should print 0
add_point(2, 4)
print(area()) # should print 8
del_point(0, 0)
del_point(0, 4)
print(closest()) # should print 16
print(area()) # should print 0
add_point(3, 4)
print(closest()) # should print 1
print(area()) # should print 0
add_point(3, 0)
print(area()) # should print 4
add_point(-100, -100)
add_point(-100, 100)
add_point(100, -100)
add_point(100, 100)
print(area()) # should print 40000
