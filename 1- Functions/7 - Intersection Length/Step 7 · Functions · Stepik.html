Intersection Length

Implement a function inter_len(l1: int, r1: int, l2: int, r2: int) -> int which takes two line segments [l1, r1] and [l2, r2] and returns the length of its intersection. You should return zero if the segments don't intersect.

Sample test cases:

print(inter_len(5, 7, 4, 6))  # should print 1
print(inter_len(1, 5, 2, 4))  # should print 2
print(inter_len(1, 7, 4, 9))  # should print 3
print(inter_len(2, 5, 6, 8))  # should print 0
